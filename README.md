## 💻 Projeto  
Esse é um projeto Web do Frontend de uma API completa que cadastra usuários e manipula notas e tags.

## 🚀 Tecnologias
Esse projeto foi desenvolvido durante o curso da Rocketseat com as seguintes tecnologias:

- ReactJS
- styled-components
- Vite
